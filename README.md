# Миний Од

2-3 настай хүүхдэд зориулсан Монгол/Англи үг сурах, дуу сурах, тоглоом, цагаан толгойн PWA demo app.

## Features

- Монгол/English хэл солих
- Үг сурах: жимс, амьтад, машин, улс орнууд, тоо
- Цагаан толгой: Монгол 35 үсэг, English alphabet
- Монгол MP3 voice assets
- Тоглоом хэсэг: зөв зураг сонгох, баяр хүргэх дуу болон алга ташилт
- iOS/Android PWA install support

## Run

Open `index.html` in a browser for local preview.

For installable PWA behavior, host this folder on HTTPS.
